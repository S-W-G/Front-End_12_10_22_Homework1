Формулы и свойства степеней

Свойства и формулы степеней используются при сокращении и
упрощении сложных выражений, при решении уравнений и
неравенств. Свойства степеней можно использовать совместно
с таблицей степеней и таблицей умножения.

a 1  = а, a 0  = 1 (a ≠ 0), a -n = 1/a n .

  • a m a n  = a m+n;
  • a m /a n  = a m-n;
  • (ab)n  = a n b n;
  • (a m ) n  = a mn;
  • (a/b) n  = a n /b n.
